# 검색 정밀도 스펙 (docs/SEARCH-PRECISION.md)

> 유사도(`src/search/searchProviders.js`의 `similarity`)가 지켜야 할 규칙과 검증 케이스.
> 이 문서가 기준이고, `test/precision-test.mjs`가 이 케이스를 자동 검증한다.
> 규칙/케이스를 바꾸면 둘 다 같이 고치고 `node test/precision-test.mjs`로 통과 확인.

## 판정 밴드 (유사도 0~1)

| 관계 | 의미 | 기대 유사도 | 근거 |
|---|---|---|---|
| **same** | 같은 상품(색상/에디션/표기만 다름) | **≥ 0.55** | verdict 비교군(≥0.45)에 확실히 포함 |
| **variant** | 다른 라인·세대·사이즈·용량 (가격대 다름) | **< 0.45** | 비교군에서 빠지거나 하위로 |
| **accessory** | 케이스/필터/부품 등 부속품 | **≤ 0.20** | `it.accessory`로 원천 배제 |
| **unrelated** | 완전 무관 | **≤ 0.15** | 시세 오염 방지 |

## 규칙 (similarity 내부, 적용 순서)

1. **가중 자카드** — 토큰 교집합/합집합. `STOP`(흔한 카테고리어·색상·에디션·해외 등)은 가중치 0.35로 낮춰 구별력 있는 토큰이 점수를 지배하게 한다.
2. **모델코드**(`modelCodes`, 하이픈/공백 무시 정규화) — 양쪽에 있고 겹치면 +0.4, 서로 다르면 **0.3 캡**.
3. **스펙**(`extractSpecs`: 용량 GB/TB, 크기 인치/cm, 부피 oz·ml·L→ml버킷, 수량) — 겹치면 +0.12, 다르면 ×0.5.
4. **변형 키워드**(`VARIANT`) — 아래 라인/세대 키워드가 한쪽에만 있으면 ×0.45. (같은 라인이면 감점 없음)
5. **부속품**(`accessoryMismatch`) — 후보에만 부속품어면 ×0.3 후 0.15 캡.

## 변형 키워드 (제품 라인·세대 — 다르면 다른 상품)

- 라인: `프로맥스/promax` · `프로/pro` · `플러스/plus` · `울트라/ultra` · `에어/air` · `미니/mini` · `맥스/max` · `라이트/lite` · `fe` · `se` · `롱/long` · `멀티/multi` · `네오/neo` · `콤보/combo` · `슬림/slim`
- 세대: `1~5세대` · `신형/구형`
- 복합 토큰(`프로맥스`)은 하위(`프로`)보다 우선 인식한다.

## 검증 케이스 (precision-test.mjs와 동기화)

### 아이폰 (라인 혼동 — 실제 신고: 프로 검색에 에어 뜸)
기준: `아이폰 15 프로 256GB`
- same → `애플 아이폰 15 프로 256GB 자급제`
- variant → `아이폰 15 프로맥스 256GB` (다른 라인)
- variant → `아이폰 15 에어 256GB` (다른 라인) ★신고 케이스
- variant → `아이폰 15 128GB` (기본형 + 다른 용량)
- variant → `아이폰 15 프로 128GB` (다른 용량)
- accessory → `아이폰 15 프로 케이스`

### 갤럭시 S (울트라/플러스/FE)
기준: `갤럭시 S24 울트라 256GB`
- same → `삼성 갤럭시 S24 울트라 256GB 자급제`
- variant → `갤럭시 S24 플러스 256GB`
- variant → `갤럭시 S24 FE 128GB`
- variant → `갤럭시 S24 울트라 512GB` (다른 용량)

### 에어팟 (세대/맥스)
기준: `에어팟 프로 2세대`
- same → `애플 에어팟 프로 2세대 정품`
- variant → `에어팟 3세대`
- variant → `에어팟 맥스`

### 스탠리 텀블러 (색상/에디션/oz↔L)
기준: `스탠리 퀜처 40oz 텀블러`
- same → `[해외]스탠리 홀리데이 데코 퀜처 텀블러 40oz Blush`
- same → `스탠리 퀜처 H2.0 플로우스테이트 40oz 텀블러 1.18L 크림`
- variant → `스탠리 퀜처 30oz 텀블러`

### 다이슨 에어랩 (컴플리트/롱)
기준: `다이슨 에어랩 스타일러 컴플리트`
- variant → `다이슨 에어랩 스타일러 컴플리트 롱`
- variant → `다이슨 에어랩 멀티 스타일러 컴플리트`

### 회귀 (기존 정상 케이스 유지)
기준: `갤럭시 버즈3 프로 SM-R630N`
- same → `삼성전자 갤럭시 버즈3 프로 SM-R630N 정품`
- variant → `삼성 갤럭시 버즈2 프로 SM-R510` (다른 모델)
- accessory → `갤럭시 버즈3 프로 실리콘 케이스`
